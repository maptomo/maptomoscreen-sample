## Key
- MaptomoAPI Key
- Google Maps Platform

Default
```
var certif = {
  'maptomo': '', // MaptomoAPI key
  'gmap': '' // Google Maps API key
};
```

## CDN
Set the specified CDN JavaScript file in the script tag of the HTML file.

Default
```
<script src="https://{CDN}" crossorigin="anonymous"></script>
```
